Eclipse Setup

Dependencies: Maven 3.x && Java 6

Installation: 
$ git clone git@github.com:redlinesmedia/redlines_client.git
$ cd redlines_client
$ mvn install

Eclipse Setup:
1) Install Maven Integration for Eclipse (m2e)
2) From Eclipse, import existing maven projects, and navigate to the local git repository.
3) Verify installation by running all JUnit tests.
